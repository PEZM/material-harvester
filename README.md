# material-harvester
 PWA for cataloguing salvaged materials on mobile
Material Harvest — a PWA for cataloging salvaged materials on mobile.
Features:

Camera integration (take photos or select from gallery)
9 categories: Doors, Windows, Timber, Metal, Brick/Stone, Glass, Fixtures, Furniture, Other
Condition tracking (Excellent → Salvageable)
Dimensions (W×H×D in mm), quantity, source location, notes
Search, filter by category, sort by date/name/condition
Data persists locally via IndexedDB (survives browser sessions)
Works offline once loaded

To install on your phone:

iPhone: Open in Safari → Share → "Add to Home Screen"
Android: Open in Chrome → Menu (⋮) → "Add to Home Screen"
